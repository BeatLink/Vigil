# Vigil

A lightweight, pluggable network and system monitor for linux



Features

* Lightweight
* Built using python
* Pull based using ssh access
* Modules for monitoring and controlling various network and system parameters
  * Data Types
    * Logging
    * Metrics
  * Collection
    * Ping / ICMP
    * SSH commands
    * HTTP/Web Apis
  * Presentation
  * Alerting
  * Control
